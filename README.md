# Apple Store Receipt Generator

A customizable Apple Store receipt generator that allows users to create professional Apple receipts with all product details, IMEI numbers, serial numbers, and export to PDF.

## Features

✅ **Complete Apple Product Database** - iPhone 12 through iPhone 17, all models and colors  
✅ **Smart Dropdown System** - Automatic model, color, and storage selection  
✅ **Automatic Specifications** - Auto-fills product specs based on selection  
✅ **Professional Receipt Format** - Apple Store authentic look and feel  
✅ **PDF Export** - Generate printable receipts in standard size  
✅ **Customizable Details** - Edit store info, customer data, prices  
✅ **Product Management** - Add/remove products with full details  
✅ **IMEI & Serial Numbers** - Track device identification numbers  

## How to Use

### Live Demo
Click here: [Live Demo](https://your-username.github.io/apple-receipt-generator/)

### Local Usage
1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. No installation required - works directly in browser

### Steps to Create Receipt:
1. **Enter Store Details** - Store name and address
2. **Set Receipt Info** - Date, time, receipt number
3. **Add Customer & Payment** - Customer name and payment method
4. **Add Products**:
   - Select product category (iPhone, iPad, Mac, etc.)
   - Choose model (iPhone 12 through 17)
   - Select color and storage
   - Enter serial/IMEI numbers
   - Set quantity
5. **Export** - Print or save as PDF
